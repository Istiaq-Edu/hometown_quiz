=== THAKURGAON - QUIZ DATA ===

Category: Places & History

[MCQ] Thakurgaon is located in which administrative division?
A) Rajshahi
B) Rangpur
C) Dhaka
D) Sylhet
CORRECT ANSWER: B) Rangpur

[TRUE/FALSE] Thakurgaon was formerly part of greater Dinajpur district.
CORRECT ANSWER: True

[MCQ] Which river flows through Thakurgaon?
A) Tangon
B) Padma
C) Meghna
D) Karnaphuli
CORRECT ANSWER: A) Tangon

[MCQ] "Baliadangi" upazila is famous for...
A) A giant Mango Tree (Surya Puri Mango Tree)
B) Tea gardens
C) Sea beach
D) Mountains
CORRECT ANSWER: A) A giant Mango Tree

[TRUE/FALSE] The Surya Puri Mango Tree is considered one of the largest mango trees in Asia.
CORRECT ANSWER: True

[MCQ] "Ranishankail" is known for...
A) Historic Zamindar Bari and Jagadal Rajbari
B) River port
C) Airport
D) Coal mine
CORRECT ANSWER: A) Historic Zamindar Bari and Jagadal Rajbari

[MCQ] "Fun City" or "Shishu Park" is a popular recreational spot in Thakurgaon town.
A) True
B) False
CORRECT ANSWER: A) True

[MCQ] Thakurgaon has an airport (though currently inactive for commercial flights).
A) True
B) False
CORRECT ANSWER: A) True

[TRUE/FALSE] Thakurgaon is a border district sharing border with India.
CORRECT ANSWER: True

[MCQ] The "Tangon River Bridge" is a notable landmark.
A) True
B) False
CORRECT ANSWER: A) True

A) Himsagar

[MCQ] The indigenous communities like Santal and Oraon live in Thakurgaon.
A) True
B) False

[MCQ] "Nekmardan Mela" is a famous traditional fair held in...
A) Rani Shankail
